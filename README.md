# RFEP Framework — archived

> **This repository has been archived.** The RFEP Framework paper is now maintained in the main FST repository:
>
> **[research-line/functional-stability-theory](https://github.com/research-line/functional-stability-theory)** → `framework/`

The Renormalized Free-Energy Principle (RFEP) establishes the mathematical foundation of Functional Stability Theory. It formalizes how renormalized energy functionals detect and resolve structural bottlenecks across mathematics, physics, and computer science — yielding a unified framework (Pattern A: Functional Positivity under Gauge Constraint) from which all domain-specific proofs derive.
